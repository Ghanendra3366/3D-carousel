# 🎡 3D Carousel Project

A visually engaging 3D carousel/slider UI created using **HTML, CSS,Python and JavaScript**. Designed for portfolios, galleries, and product showcases.

## 🚀 Features
- 3D rotating carousel effect
- Smooth transitions
- Responsive layout
- Lightweight and customizable

## 🛠️ Tech Stack
- HTML5
- CSS3 (3D Transforms)
- JavaScript (Vanilla)
